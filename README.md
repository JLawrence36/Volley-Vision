# VolleyVision

**Film in. Stats out.**

VolleyVision is a volleyball video breakdown app built with Expo React Native.

## What it does

- Add players to a roster
- Upload/select volleyball game film
- Watch video inside the app
- Tap stats while watching film
- Assign stats to players
- Save timestamped stat events
- View player totals
- View game and season reports
- Jump back to tagged plays

## Current MVP Features

- Roster management
- Game video selection
- Manual stat tagging
- Player assignment
- Timestamped stat log
- Game reports
- Season reports
- Local device storage

## Tech Stack

- Expo
- React Native
- expo-video
- expo-document-picker
- AsyncStorage
- EAS Build
